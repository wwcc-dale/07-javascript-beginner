Level 2 – Stretch (3 points)

Challenge 5: Timer Class

Create a Timer class with: - Constructor takes seconds - Method tick() decreases seconds by 1 - Method isFinished() returns true when seconds === 0 - Method reset(newSeconds) sets seconds to a new value

Challenge 6: Multiple Instances

Create 3 different Student instances and use a loop to: - Print each student's name - Check if they're passing - Count how many students are passing

Level 3 – Advanced (2 points)

Challenge 7: Grade Book System

Create a GradeBook class that: - Has an array of Student instances - Method addStudent(student) adds to the array - Method getPassingCount() returns how many students are passing - Method getAverageGrade() returns the average of all student grades - Method getTopStudent() returns the student with the highest grade

Test with at least 4 students with different grades.
