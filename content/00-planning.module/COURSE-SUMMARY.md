## Module 01: JavaScript Foundations (Sessions 1-5)

Learn the fundamentals of JavaScript programming.

### Sessions
- **Session 1:** Variables, Data Types, Operators
- **Session 2:** Control Flow (if/else, loops)
- **Session 3:** Functions and Scope
- **Session 4:** Arrays and Array Methods
- **Session 5:** Objects and Properties

### Key Topics
- Variables (let, const, var)
- Primitive vs reference types
- Control structures
- Function declarations and arrow functions
- Array manipulation (map, filter, reduce)
- Object creation and access
- Scope and closures

### Skills Earned
✅ Write JavaScript programs
✅ Manipulate data with arrays and objects
✅ Control program flow
✅ Create reusable functions
✅ Understand scope and context
