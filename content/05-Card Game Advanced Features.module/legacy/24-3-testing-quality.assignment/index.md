# Session 24: Write Tests and Documentation

Write unit tests and add professional documentation to code.

**Time:** 60–90 minutes | **Points:** 20

## Level 1 – Core (15 pts)

- Unit tests for Card class (10+ tests)
- Unit tests for Deck class (10+ tests)
- Unit tests for Game class (5+ tests)
- JSDoc comments on all classes/methods
- Test output shows pass/fail for each
- No console errors

## Level 2 – Stretch (3 pts)

- Tests for edge cases (empty deck, invalid cards, etc.)
- Code refactoring for clarity
- README.md with instructions

## Level 3 – Advanced (2 pts)

- Comprehensive test suite (50+ tests)
- Code coverage report
- Advanced refactoring
