# Session 13: Dynamic Styling and Themes

Use classList and style manipulation to build interactive UIs.

**Time:** 60–90 minutes | **Points:** 20 | **Submit:** `13-assignment.html`

## Getting Started

- buttons
- Download Starter File | 13-starter.html | primary | download

1. Open the starter file, save a copy as `13-assignment.html`, and complete each challenge.

## Level 1 – Core [15 pts](#pill:accent)

### Challenge 1: Class Toggle System
- 3 boxes with default styling
- Buttons: "Highlight", "Hide", "Show", "Reset"
- Use classList.add/remove/toggle appropriately

### Challenge 2: Theme Switcher
- Button "Toggle Dark Mode"
- Clicking adds/removes .dark-mode class to body
- Define dark-mode styles in CSS (bg, text colors)

### Challenge 3: Interactive Cards
- 3 cards, each with "Expand" button
- Clicking toggles .expanded class (changes width/height)
- Button text changes "Expand"/"Collapse"

### Challenge 4: Status Indicator
- Dropdown with options: success, warning, error, info
- Selecting updates div with appropriate class and message
- Each class has unique styling

## Level 2 – Stretch [3 pts](#pill:cert)

### Challenge 5: Filter System
Buttons for categories, clicking filters list items by adding/removing classes

### Challenge 6: Style Customizer
Inputs for color, font-size, border-radius that update element styles live

## Level 3 – Advanced [2 pts](#pill:degree)

### Challenge 7: Complete Theme System
Multiple themes (light, dark, blue, green), save preference, apply to all elements
