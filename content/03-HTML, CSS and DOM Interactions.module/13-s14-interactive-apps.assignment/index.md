# Session 14: Interactive Applications

Build fully interactive applications with event handling.

**Time:** 60–90 minutes | **Points:** 20 | **Submit:** `14-assignment.html`

## Getting Started

**[Download starter file](14-starter.html)**

1. Open the starter file, save a copy as `14-assignment.html`, and complete each challenge.

## Level 1 – Core (15 pts)

### Challenge 1: Counter App
- Display showing count (starts 0)
- Buttons: +1, -1, +5, -5, Reset
- Each button updates count

### Challenge 2: Form Handler
- Form: name input, email input, submit button
- On submit: prevent default, display welcome message, clear form
- Validate: both fields required

### Challenge 3: Interactive List
- Input + "Add" button
- Clicking adds item to list
- Each item has delete button
- Empty input does nothing

### Challenge 4: Color Picker
- 6 color buttons
- Clicking changes box background to that color
- Display current color name

## Level 2 – Stretch (3 pts)

### Challenge 5: Calculator
Basic calculator with number buttons, operator buttons, display

### Challenge 6: Character Counter
Textarea that shows character count live as you type

## Level 3 – Advanced (2 pts)

### Challenge 7: Complete Todo App
Add todos, mark complete, filter (all/active/completed), delete, clear completed
