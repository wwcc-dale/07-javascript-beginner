## Study Guide (30 minutes)

### Key Concepts to Review

**JavaScript Basics:**
- Declaring variables (const, let)
- Data types (string, number, boolean, array, object)
- Comparison operators (===, !==, <, >)
- Logical operators (&&, ||, !)
- if/else statements
- for loops and while loops
- Function declaration and calling
- Parameters and return values

**Arrays:**
- Creating arrays
- Accessing elements (index)
- Array methods: push, pop, shift, unshift, length
- Looping through arrays
- Finding elements

**Objects:**
- Creating objects
- Accessing properties (dot and bracket notation)
- Adding/modifying properties
- Methods in objects
- this keyword

**Classes:**
- Class syntax and constructor
- Creating instances with new
- Instance properties and methods
- Encapsulation with _ convention
- Getters and setters
- Working with arrays of instances
- Inheritance with extends
- super() constructor call
- Method overriding

**Node.js:**
- Running JavaScript outside browser
- npm package manager
- Installing packages
- Importing/exporting modules (CommonJS and ES6)
- package.json structure

**DOM Manipulation:**
- document.querySelector('#id')
- document.querySelectorAll('.class')
- element.textContent vs element.innerHTML
- element.getAttribute() and element.setAttribute()
- document.createElement()
- element.appendChild()
- element.remove()

**CSS with JavaScript:**
- element.style.property
- element.classList.add()
- element.classList.remove()
- element.classList.toggle()
- element.classList.contains()

**Events:**
- element.addEventListener('event', function)
- Common events: click, input, change, submit
- event.preventDefault()
- event.target
- Reading form input values

5. **Arrays** - Manipulate arrays with methods
6. **Objects** - Access and modify object properties
7. **Classes** - Write class definitions
8. **DOM Selection** - Select elements from HTML
9. **DOM Modification** - Change content and attributes
10. **Event Handling** - Attach listeners and handle events
