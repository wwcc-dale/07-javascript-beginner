# Session 22: Add Data Persistence

Implement Local Storage for statistics, game history, and settings.

**Time:** 60–90 minutes | **Points:** 20

## Level 1 – Core (15 pts)

- GameStats class (load/save win/loss/tie/score)
- Display statistics in game UI
- Persist across page refreshes
- Clear stats button with confirmation
- Save player name preference

## Level 2 – Stretch (3 pts)

- Game history tracking (save 50 recent games)
- Recent games display
- Filter history by outcome

## Level 3 – Advanced (2 pts)

- Settings persistence (theme, difficulty, sound)
- Auto-load preferences on page load
- Export/import game data
